# MusicDB
IMDb de la musique en Flask + MySQL.
Fonctionnalités :
- OOP (héritage, polymorphisme)
- Reviews + étoiles
- Login / Register
- Recherche
- Top 10 par genre

pour lancer l'app:
lancer import_lastfm.py et import_deezer.py pour les differentes ressources api
En suite :
-pour l'app web (flask) : app.py
-pour l'app dekstop (tkinter) : tk_app.py
